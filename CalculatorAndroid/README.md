# ماشین‌حساب اندروید

## روش پیشنهادی: ساخت APK رایگان با GitHub Actions (بدون نیاز به دانلود SDK)

چون دانلود مستقیم Android SDK از داخل ایران فیلتر است، در این پروژه یک فایل
خودکارسازی (`.github/workflows/build-apk.yml`) گذاشته شده که کار ساخت APK را
روی سرورهای رایگان گیت‌هاب (که فیلتر نیستند) انجام می‌دهد.

### مراحل:

1. یک حساب رایگان در [github.com](https://github.com) بساز (اگر نداری).
2. یک ریپازیتوری (repository) جدید بساز — مثلاً به اسم `calculator-android`.
   هنگام ساخت، گزینه **Public** یا **Private** هر دو مشکلی ندارند.
3. کل محتوای پوشه `CalculatorAndroid` (همین پوشه‌ای که الان داری) را در آن
   ریپازیتوری آپلود کن. ساده‌ترین راه:
   - وارد صفحه ریپازیتوری در مرورگر شو.
   - دکمه **Add file → Upload files** را بزن.
   - تمام فایل‌ها و پوشه‌ها (از جمله پوشه مخفی `.github`) را بکش و رها کن.
   - دکمه **Commit changes** را بزن.

   (اگر با خط فرمان git راحت‌تری: `git init`, `git add .`, `git commit -m "init"`,
   و بعد `git remote add origin <لینک‌ریپو>` و `git push -u origin main`)

4. به تب **Actions** در صفحه ریپازیتوری برو. خودش به‌طور خودکار شروع به اجرا
   می‌کند (یک دایره زرد/سبز کنار کامیتت می‌بینی). حدود ۲ تا ۴ دقیقه طول می‌کشد.
5. وقتی تیک سبز ✅ زده شد، روی همان اجرا (workflow run) کلیک کن؛ پایین صفحه
   بخش **Artifacts** را می‌بینی با اسم `calculator-apk` — آن را دانلود کن.
   یک فایل zip است که داخلش `app-debug.apk` قرار دارد.
6. فایل APK را روی گوشی اندرویدی منتقل کن و نصبش کن (ممکن است لازم باشد گزینه
   "نصب از منابع ناشناس" را در گوشی فعال کنی).

نکته: اگر دسترسی به github.com هم برایت مشکل دارد، با یک VPN ساده فقط برای
چند دقیقه‌ی آپلود کافی است — خودِ ساخت APK روی سرور گیت‌هاب انجام می‌شود، نه
روی کامپیوتر تو.

## روش جایگزین: ساخت APK محلی در Android Studio (نیاز به VPN دارد)

1. [Android Studio](https://developer.android.com/studio) را نصب کن.
2. یک VPN را روشن کن (چون دانلود Android SDK از سرورهای گوگل است).
3. در Android Studio گزینه **Open** را بزن و پوشه `CalculatorAndroid` را انتخاب کن.
4. صبر کن تا Gradle خودش وابستگی‌ها را دانلود و پروژه را Sync کند.
5. از منو بالا: **Build → Build Bundle(s) / APK(s) → Build APK(s)** را بزن.
6. فایل APK داخل پوشه `app/build/outputs/apk/debug/app-debug.apk` قرار می‌گیرد.
